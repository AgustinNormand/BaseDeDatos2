Incluir la libreria <json-c/json.h>

Crear un struct json_object para el archivo
Crear un struct json_object para cada valor que quiera parsear del archivo.

Pasarle un buffer leido del archivo a json_tokener_parse y asignarlo al json_object.

json_object_get_ex("json_object","NombreClave","json_object para el valor");

Obtener el valor
json_object_get_string("json_object para el valor");

Para compilar el .c agregar -ljson-c

Ejemplo:
struct json_object *parsed_json
struct json_object *name;

parsed_json = json_tokener_parse(buffer);

json_object_get_ex_(parsed_json, "Nombre", &name);

printf("Name: %s\n",json_object_get_stirng(name));
